# 副業マッチングアプリ

## 概要
フリーランスと企業をマッチングし、案件応募・契約・チャットまでを一元管理できるWebアプリです。

---

## 背景・課題
プロジェクトにおける人材配置や業務分担は、スキルや適性を十分に考慮できていないケースも多く、
適切なマッチングが行われないことによる非効率が課題となっている。
特に、タスクの偏りや工程間の分断により、生産性の低下やコミュニケーションコストの増加が発生していた。
この課題を踏まえ、案件・応募・契約までを一元管理できるマッチングシステムの設計を行った。

---

## 機能

### ユーザー機能
- ユーザー登録（ワーカー / 企業）
- ログイン / 認証
- プロフィール管理

### 案件機能
- 案件投稿
- 案件一覧表示
- 案件応募
- 採用（マッチング）

### 契約・運用機能
- 契約管理（Contract）
- チャット機能
- レビュー機能

---

## 技術スタック
- Python
- Django（ORM / Authentication）
- SQLite
- HTML / CSS

---

## 工夫した点
- Job → Application → Contract の業務フロー設計
- max_workersによる採用制御
- seedコマンドによるデータ初期化（環境再現性）
- roleによる権限制御（worker / company）
- 状態遷移（応募→採用→契約）の設計

---

## 苦労した点
- マッチングと契約の状態管理設計
- シグナルやseedデータとの整合性調整
- データリセット時の冪等性確保
