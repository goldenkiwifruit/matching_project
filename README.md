# 副業マッチングアプリ

## 概要
フリーランスと企業をマッチングし、案件応募・契約・チャットまで行えるWebアプリ

---

## 機能
- ユーザー登録（ワーカー / 企業）
- 案件投稿
- 案件応募
- マッチング機能（応募 → 採用）
- 契約管理（Contract）
- チャット機能
- レビュー機能
- seedデータで初期化可能

---

## 技術スタック
- Python
- Django
- SQLite
- HTML / CSS

---

## 工夫した点
- Job → Application → Contractの業務フロー設計
- max_workersによる採用制御
- seedコマンドによる環境再現
- roleによる権限制御（worker / company）

---

## 苦労した点
- マッチングと契約の状態管理
- シグナルとseedの整合性
- データリセットの冪等性
